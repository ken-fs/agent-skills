## 排版说明

### 标题层级
- H2 (##) - 主要章节
- H3 (###) - 次级小节
- 避免使用H1，由公众号文章标题承担

### 金句格式
```markdown
> ✨ 金句内容
```
使用引用块 + ✨符号高亮

### 图片格式
```markdown
![](图片URL)
*图片说明*
```

### 代码块
```markdown
​```语言
代码内容
​```
```

### 强调
- **加粗**：重要概念
- *斜体*：图片说明、引用来源

### 列表
- 无序列表：-
- 有序列表：1. 2. 3.

### 分隔线
```markdown
---
```
用于章节间分隔

### 引用
```markdown
> 引用内容
```

### 链接
```markdown
[链接文字](URL)
```

## 公众号编辑器适配注意事项

1. **避免复杂Markdown语法**
   - 不用嵌套列表
   - 不用表格（改用文字列表）
   - 不用HTML标签

2. **图片处理**
   - 上传到公众号后台
   - 图片宽度建议900px
   - 配上说明文字

3. **代码块**
   - 使用公众号代码块功能
   - 或使用代码高亮工具（如mdnice）

4. **格式检查**
   - 段落间距适中（2个换行）
   - 标题层级清晰
   - 重点内容加粗

5. **最终检查**
   - 在公众号编辑器预览
   - 手机端查看效果
   - 调整间距和字体

---

## 复制到公众号的步骤

1. **方法1: 直接粘贴**
   - 复制Markdown内容
   - 粘贴到公众号编辑器
   - 手动调整格式

2. **方法2: 使用Markdown编辑器**
   - 使用[mdnice](https://mdnice.com/)等工具
   - 粘贴Markdown内容
   - 选择样式主题
   - 一键复制到公众号

3. **方法3: 使用脚本转换**
   - 使用wechat_formatter.py
   - 转换为公众号优化格式
   - 复制到编辑器

推荐使用方法2，最省时高效。
