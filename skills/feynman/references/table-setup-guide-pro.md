## ⚙️ 字段详细配置

### 1. 概念（单行文本）
- **字段类型**: 单行文本
- **示例**: "React Hooks", "TCP三次握手", "快速排序算法"

### 2. 分类标签（多选）
- **字段类型**: 多选
- **选项配置**:
  ```
  前端开发
  后端开发
  算法与数据结构
  数据库
  网络协议
  操作系统
  DevOps
  云计算
  架构设计
  AI与机器学习
  移动开发
  安全
  测试
  工具与效率
  其他
  ```
- **颜色建议**: 为不同分类设置不同颜色便于视觉识别

### 3-8. 文本字段（多行文本）
- **字段类型**: 多行文本
- **启用富文本**: 建议开启，支持 Markdown 格式

### 9. 掌握程度（评分）
- **字段类型**: 评分
- **最大分值**: 5星
- **评分说明**:
  - ⭐ (1星): 刚接触，基本不理解
  - ⭐⭐ (2星): 有基本概念，但不能应用
  - ⭐⭐⭐ (3星): 理解核心概念，能简单应用
  - ⭐⭐⭐⭐ (4星): 深入理解，能独立解决问题
  - ⭐⭐⭐⭐⭐ (5星): 精通，能够教授他人

### 10. 学习日期（日期）
- **字段类型**: 日期
- **格式**: ✅ 勾选 "包含时间"
- **默认值**: 自动填充当前时间

### 11. 下次复习（日期）
- **字段类型**: 日期
- **复习间隔规则**（基于遗忘曲线）:
  - 第1次复习: 学习后 1 天
  - 第2次复习: 学习后 3 天
  - 第3次复习: 学习后 7 天
  - 第4次复习: 学习后 15 天
  - 第5次复习: 学习后 30 天
  - 第6次复习: 学习后 60 天

### 12. 相关概念（关联字段）
- **字段类型**: 关联字段
- **关联表**: 当前表（自关联）
- **关联字段**: 选择 "概念" 字段
- **允许关联多条**: ✅ 开启

### 13. 学习时长（数字）
- **字段类型**: 数字
- **单位**: 分钟
- **小数位数**: 0

### 14. 完成状态（单选）
- **字段类型**: 单选
- **选项配置**:
  ```
  🟡 学习中
  🟢 已掌握
  🔵 需复习
  🟠 待深入
  ⚪ 已归档
  ```

### 15. AI关键词（多选）
- **字段类型**: 多选
- **说明**: 由 AI 自动提取的技术关键词
- **自动生成**: 脚本会自动从学习内容中提取

### 16. 思维导图（URL）
- **字段类型**: URL
- **示例**: XMind、MindMeister、ProcessOn 等工具生成的分享链接

### 17. 待深入问题（多行文本）
- **字段类型**: 多行文本
- **用途**: 记录学习过程中发现的、需要进一步研究的深层次问题

## 🚀 快速配置步骤

### Step 1: 创建基础字段（5分钟）
```bash
1. 打开你的飞书多维表格
2. 逐一添加字段 1-10（必需字段）
3. 确保字段名称完全一致（包括中文）
```

### Step 2: 配置多选和单选选项（3分钟）
```bash
1. "分类标签" 字段 → 添加 15 个选项
2. "完成状态" 字段 → 添加 5 个选项
3. 为选项设置不同颜色
```

### Step 3: 添加进阶字段（可选，5分钟）
```bash
1. 添加字段 11-17
2. 配置 "相关概念" 自关联
3. 设置 "AI关键词" 多选字段
```

### Step 4: 创建视图（5分钟）
```bash
1. 复制默认视图，重命名为 "待复习"
2. 设置筛选条件
3. 重复创建其他视图
```

## 🔧 常见问题

### Q1: 如何批量修改字段名称？
A: 不建议批量修改。建议按照本文档逐个创建，确保名称完全一致。

### Q2: 关联字段如何配置？
A: "相关概念" 字段配置步骤：
1. 字段类型选择 "关联"
2. 选择 "当前表"
3. 显示字段选择 "概念"
4. 勾选 "允许关联多条记录"

### Q3: 日期字段时区问题
A: 确保飞书账号时区设置正确，脚本会使用本地时区。

### Q4: 能否自定义复习间隔？
A: 可以！在脚本中修改 `REVIEW_INTERVALS` 配置即可。
