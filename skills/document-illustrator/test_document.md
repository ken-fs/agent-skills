# Document Illustrator 测试文档

这是一个用于测试 Document Illustrator skill 的示例文档。

## AI 编程助手的演化

人工智能编程助手正在快速发展，从简单的代码补全到复杂的代码生成和理解。

### Claude Code 的特点

Claude Code 是一个强大的编程助手，具有以下特点：
- 智能代码理解和生成
- 多语言支持
- 上下文感知能力
- 与开发工具的集成

### Antigravity 代理

Antigravity 是一个本地代理服务，可以：
- 提供统一的 API 接口
- 支持多个 AI 模型
- 本地化部署，保护隐私
- 降低 API 调用成本

## Document Illustrator Skill

这个 skill 可以为文档自动生成配图。

### 核心功能

- 智能分析文档结构
- 多种风格选择（渐变玻璃、票据、矢量插画）
- 支持自定义比例和分辨率
- 批量生成配图

### 技术实现

使用 Gemini API 的图片生成功能，通过精心设计的提示词模板，生成符合文档主题的高质量配图。
